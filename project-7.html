Hack!T Organizer

A mobile app for Hack!T Organizers of the student club CSE (Club Scientifique de l'ESI)

Code Link (Hack!T Organizer)

Project Overview

Hack!T Organizer is a mobile app for Hack!T organizers. Hack!T is the biggest hackthon organized by students in Algeria and it's organized by CSE (Club Scientifique de l'ESI). This app was made for its organizers, it shows the agenda for the actual day, a page for the organizer where he can find his tasks day by day, he can check the participants list in an activity and check their qr code for this activity.

Tools Used

Dart
Flutter
GIT
Figma
